Carnival SDK Documentation

The Carnival SDK for Unity recognizes the finger, hand and gestures from a depth map produced by a depth sensor and allows you to use that data to interact inside a virtual scene.

Overview

  • The Controller class is the main interface to the Carnival SDK. Create an instance of this Controler class to access frames of tracking data and configuration information.

Following classes in the Carnival namespace define the Carnival tracking objects:

  • A Frame is a set of data recorded at a specific moment in time. A Frame contains Hand objects.
  • A Hand object represents a tracked hand. A Hand has Finger objects and properties like CoG (Center of Gravity), Handedness, Openness etc.
  • A Fingertip object represents a tracked fingertip or thumb tip of a hand.

Following classes in the Carnival namespace define the Carnival Gesture objects and events:

  • Clamp
  • Swipe

Prerequisites

Before you begin working with Carnival SDK, make sure your system meets the following minimum hardware and software requirements:

General

  • Unity 5.3.x (32- or 64 Bit)
  • pmdtechnologies CamBoard pico flexx sensor

Platform Specific

  • Android development
    • Android 5.x (Jelly Bean) or higher
    • Additional Unity package for Android build support
    • Android SDK
    • Java SE Development Kit 8
    • Samsung Galaxy S7/S7 edge
    • Cardboard SDK for Unity (Ver. 0.7.0) is recommended
    • Developing on Windows: Samsung Android USB Driver for Windows
  • Mac development
    • OS X 10.11 (El Capitan) or higher
    • Additional Unity package for Mac build support
  • Windows development
    • Microsoft Windows 7/8/10 (32- or 64 Bit)
    • Additional Unity package for Windows build support
    • CamBoard pico flexx driver installed

Installation

Import the CarnivalSDK.unitypackage into your Unity project and you are ready to go.

Getting Started

You need to write your own scripts to access the data from the Carnival SDK API. All Carnival SDK classes are defined in the Carnival namespace. A basic MonoBehavior class that uses the Carnival SDK API will look like the following:

using UnityEngine;
using Carnival;
public class FingertipBehaviour : MonoBehaviour
{
Controller carnivalController;
void Start()
{
carnivalController = new Controller();
// Initalize the controller which will try to connect to the sensor.
carnivalController.Init();
// Starts hand detection and gesture recognition.
carnivalController.Start();
}
void Update()
{
Frame frame = carnivalController.Frame();
foreach (Hand hand in frame.Hands)
{
foreach (Fingertip fingertip in hand.Fingertips)
{
// Now do something cool with the detected fingertip data in the frame.
}
}
}
void OnDestroy()
{
// Stops hand detection and gesture recognition.
carnivalController.Stop();
}
}

Examples

The Carnival SDK samples pack which is included in this package demonstrates various features of the SDK and how they can be used, such as fingertip tracking, getting hand properties, gesture recognition etc. It consists of several demo scenes for different use cases. These sample scenes will help you getting started utilizing the Carnival SDK in your own project.

How to Run the Examples:

  1. Import CarnivalSDKSamples.unitypackage
  2. Open and run one of the provided scenes
