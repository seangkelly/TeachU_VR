Detailed Description

The Hand class stores the tracking results of a single hand.

Property Documentation

Vector3 CenterOfGravity
getset

Reports the center of gravity of the hand.

The center position of the hand is given in meters from the sensor origin.

float Confidence
getset

How confident is the detection that a hand was detected.

The confidence level ranges between 0.0 and 1.0 inclusive.

float Openness
getset

Property holding the amount of openness of the hand.

The openness level ranges between 0.0 (closed) and 1.0 (open) inclusive.
