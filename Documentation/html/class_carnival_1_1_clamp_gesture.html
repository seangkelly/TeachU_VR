Detailed Description

The ClampGesture class represents a recognized clamp or pinch gesture.

The clamp gesture is recognized when the tips of the thumb and the index finger touch such that the two fingers form an ellipsoid. When a clamp gesture is recognized the midpoint of the ellipsoid is computed.
